# PHP Login System

This is a basic PHP login system created for practice and interview preparation.

## Features
- User login
- User signup (registration)
- Logout functionality
- Session based authentication
- Welcome page after login
- Partial files used for navbar and database connection

## Folder Structure
- `log-in.php` – User login page
- `sign_up.php` – User registration page
- `log-out.php` – Logout logic
- `welcome.php` – Welcome page after successful login
- `partils/nav-bar.php` – Navbar partial file
- `partils/userdb.php` – Database connection file

## Technologies Used
- PHP
- MySQL
- HTML
- CSS

## Note
This project is created for learning purpose.  
Database credentials are kept separate for security.
